Social Media API

Authentication Endpoints Security

Build Something Amazing

A powerful and secure RESTful API for building modern social media applications with user authentication, posts, and media upload capabilities.

Secure Authentication

JWT-based authentication with bcrypt password hashing for maximum security.

User Management

Complete user lifecycle management with profile customization.

Media Upload

Cloudinary integration for seamless image and video uploads.

Base URL

https://express-auth-so6r.vercel.app

Authentication

Include the JWT token in the Authorization header:

Authorization: Bearer <your_jwt_token>

Endpoints

Authentication

POST /auth/signup Create a new user account

Request Body:

{
  "username": "johndoe",
  "password": "securepass123",
  "email": "john@example.com",
  "avatar": "(file - optional)"
}

Response:

{
  "token": "eyJhbGciOiJIUzI1NiIs...",
  "user": {
    "id": "user_12345",
    "username": "johndoe",
    "email": "john@example.com",
    "avatar": "https://..."
  }
}

Request Body:

Users

GET /users Get all users

Response:

[
  {
    "id": "user_12345",
    "username": "johndoe",
    "email": "john@example.com",
    "avatar": "https://..."
  },
  {
    "id": "user_67890",
    "username": "janedoe",
    "email": "jane@example.com",
    "avatar": "https://..."
  }
]
GET /users/:username Get user by username

Response:

{
  "id": "user_12345",
  "username": "johndoe",
  "email": "john@example.com",
  "avatar": "https://...",
  "posts": [
    {
      "id": "post_12345",
      "title": "Hello World",
      "content": "My first post!"
    }
  ]
}
PUT /users/upd/:id Update user profile

Request Body:

{
  "username": "johndoe_updated",
  "email": "john.new@example.com",
  "avatar": "(file - optional)"
}

Response:

{
  "id": "user_12345",
  "username": "johndoe_updated",
  "email": "john.new@example.com",
  "avatar": "https://..."
}

Posts

POST /posts/new Create new post

Request Body:

{
  "title": "My First Post",
  "content": "Hello everyone!",
  "media": "(file - optional)"
}

Response:

{
  "id": "post_12345",
  "title": "My First Post",
  "content": "Hello everyone!",
  "media": "https://...",
  "author": {
    "id": "user_12345",
    "username": "johndoe"
  },
  "createdAt": "2025-03-15T14:30:00Z"
}
