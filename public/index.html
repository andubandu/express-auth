Social Media API

Authentication Endpoints Security

Build Something Amazing

A powerful and secure RESTful API for building modern social media applications with user authentication, posts, and media upload capabilities.

Secure Authentication

JWT-based authentication with bcrypt password hashing for maximum security.

User Management

Complete user lifecycle management with profile customization.

Media Upload

Cloudinary integration for seamless image and video uploads.

Base URL

https://express-auth-so6r.vercel.app

Authentication

Include the JWT token in the Authorization header:

Authorization: Bearer <your_jwt_token>

Likes

GET /likes Fetches ALL likes, you can either filter by post id (https://express-auth-so6r.vercel.app/likes?postId=yourpostid, shows likes on post), by user id (https://express-auth-so6r.vercel.app/likes?userId=youruserid, shows likes by user) or just get the count of likes (/likes?postId=postId&number=0), atleast ONE param is required

Response:

{"likes":[]}
POST /likes/:postId Like or dislike a post (based off if you have liked it or not)

Response:

                  // if i haven't like the post
                  {
                    "message": "Liked post successfully",
                    "like": {
                      "_id": "newLikeId",
                      "userId": "userId",
                      "postId": "postId",
                      "createdAt": "date",
                      "deletedAt": null
                    }
                  }
                  // if i have liked the post
                  {
                    "message": "Post unliked successfully"
                  }
